# rock-paper-scissors-classification
Repository ini berisi model klasifikasi yang dibuat untuk menebak gambar rock paper scissors.

Data yang digunakan adalah data dengan author name Julien de la Bru√®re-Terreault (drgfreeman@tuta.io) yang diperoleh dari [kaggle](https://www.kaggle.com/drgfreeman/rockpaperscissors), klik [disini](https://www.kaggle.com/drgfreeman/rockpaperscissors/download) untuk download data. 

Dataset berisi 2188 gambar yang mana terdiri dari gestur tangan 'Rock' (726 gambar), 'Paper' (710 gambar) dan 'Scissors' (752 gambar).
